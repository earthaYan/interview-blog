# React 19 新特性

## 自动批处理

批处理价值：连续多次 setState，合并更新最终只渲染一次

1. 减少不必要的渲染
2. 避免界面闪烁和卡顿
3. 降低 CPU 计算开销

React18 以前：只在 React 事件处理器里面支持批处理

### flushSync

:::danger
不推荐使用
:::
如果需要立刻同步执行状态更新，跳过批处理，强制执行渲染

```js
flushSync(() => {
  setState({ name: 1 });
});
```
