# 浏览器工作流程

浏览器从输入 url 到呈现页面主要分成 2 个阶段：加载资源和页面渲染

## 资源加载

1. DNS 解析：将域名解析为真实的 ip 地址
2. 浏览器向服务器发起 http 请求
3. 服务器处理请求并返回资源给浏览器

## 页面渲染

> DOM 和 CSSOM 属于结构化对象

1. 根据 html 代码生成 DOM 树
2. 根据 css 代码生成 CSSOM
3. 结合 CSSOM+DOM 树生成 Render tree
4. 浏览器根据 render 树进行渲染
5. 遇到 script 则暂停渲染，优先加载并执行 js
6. 执行完继续渲染，直至完成整个 render tree 的渲染
