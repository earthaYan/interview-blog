# 前端安全

常见的攻击：XSS,CSRF

XSS 攻击是跨站脚本攻击，指攻击者将恶意脚本注入网页，当其他用户访问该页面时，这些脚本会在用户浏览器中执行，从而可能窃取用户信息（如 Cookie、Token）、篡改页面内容或进行其他恶意操作。XSS 包括反射型、存储型和 DOM 型。

常见的解决方案是：

1. 对用户输入进行校验
2. 前后端同时替换特殊字符(可自行替换也可使用 npm 包 xss)
3. 使用 CSP 策略，在服务器配置文件中添加 CSP 响应头或者后端设置

---

CSRF 是指攻击者通过非常规手段让用户以登录状态，在不知情的情况下发起恶意请求，从而执行未授权操作

常见解决方案：

1. 使用 post 方法
2. 增加二次验证
3. 在每次表单提交或 API 请求中加入服务端随机生成的 token，服务器验证 token 是否匹配
