# CDN 内容分发网络

由各个服务器节点组成，将网站内容更靠近客户，让客户访问更快

- 源站存储网站原始内容
- CDN 节点负责缓存和分发内容
- 智能调度系统决定用户访问哪一个节点

## 核心机制

1. 就近访问，减少物理距离带来的延迟
2. 减轻源站压力，静态资源由 CDN 处理，源站只需应对动态业务
3. 优化资源和带宽利用，CDN 服务商拥有高质量的网络基础设施,支持突发流量，多协议优化(http2,Quic)提升了传输效率,最终用户感知到的是加载更快
4. 智能调度提升可用性和容错性，内置负载均衡和自动容错机制，节点异常会自动切换
