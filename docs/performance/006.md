# 图片资源优化

## 优化意义

1. 图片是体积最大的静态资源之一
2. 优化图片可以减少页面首屏加载时间
3. 图片性能会直接影响 SEO、转化率和留存率
4. 优化是一种系统性工程，需要多维度衡量

## 格式选择

- 新格式：`avif`,`webP`,压缩率高，体积小
- 传统格式:`jpeg`,`png`，png 适合透明图标后者 Logo，jpeg 适合照片

## 响应式图片

使用`srcset`属性按需提供最合适的尺寸

```html
<img
  srcset="image-480w.jpg 480w, image-800w.jpg 800w"
  sizes="(max-width:600px) 480px, 800px"
  src="image-800w.jpg"
  alt="响应式图片"
/>

<picture>
  <source srcset="img.webp" type="image/webp" />
  <source srcset="img.jpg" type="image/jpeg" />
  <img src="img.jpg" />
</picture>
```

> 总结：格式层面优先选新格式；加载层面利用 srcset 和懒加载；压缩层面在构建阶段自动化处理；架构层面接入 CDN 和图片服务
