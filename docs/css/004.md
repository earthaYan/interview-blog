# 垂直居中实现方案

## flex 布局

```css
.parent {
  display: flex;
  align-items: center; /* 交叉轴垂直居中 */
}
```

## grid 布局

```css
.parent {
  display: grid;
  align-items: center;
  height: 300px;
  place-items: center;
}
```

## 绝对定位+transform

```css
.parent {
  position: relative;
}
.box {
  position: absolute;
  /* 将 .box 的顶部定位在父容器高度的 50% 处
  此时.box 的顶部位于父容器的中线。
   */
  top: 50%;
  /* 把 .box 自身向上移动自身高度的一半， */
  transform: translateY(-50%);
}
```
