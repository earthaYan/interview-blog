# Flex 与 Grid 布局讲解

## 🧩Flex 布局

> **面试官常问**：  
> “能讲讲 Flex 布局吗？你在项目中一般怎么用？”

### 思路回答

Flex 布局主要用于**一维空间**上的对齐和分布。我理解他是**主轴方向控制 + 对齐规则组合**。一个容器设置 `display: flex` 之后，它就变成一个 **flex 容器（container）**，  
容器里的直接子元素就是 **flex items**。

### 主轴与交叉轴

通过 `flex-direction` 决定主轴方向：

```css
flex-direction: row | row-reverse | column | column-reverse;
```

主轴方向影响 `justify-content`，交叉轴影响 `align-items`。

### 常用属性讲解

- justify-content\*\*：主轴对齐方式
- align-items：单行的交叉轴对齐
- align-content：多行的交叉轴对齐
- flex-wrap：是否允许换行
- gap：定义项目间距
- flex-grow / flex-shrink / flex-basis：控制子元素伸缩性

### 示例：导航栏布局

```css
.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 0 20px;
}
```

### 实际项目经验补充

经常在项目中用 Flex 来做：

- 顶部导航栏 / 工具栏布局
- 按钮组或标签排列

## 🧱 Grid 布局

> **面试官常问**：  
> “那 Grid 呢？你用过吗？和 Flex 有什么不同？”

### 思路回答

Grid 是专门为**二维布局**设计的，也就是**同时控制行和列**。Flex 更偏向“一条轴”，而 Grid 能建立整个页面的网格系统。

### 基础定义

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 100px;
  gap: 10px;
}
```

### 子元素控制

```css
.item1 {
  grid-column: 1 / 3;
  grid-row: 1 / 2;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}
```

### Grid 的常见应用场景

- 仪表盘、多面板布局
- 图表 + 控制区混合布局
- 等比例自适应卡片区
- 多维数据展示（例如地图 + 列表）

## ⚖️ Flex vs Grid 对比总结

| 对比点   | Flex             | Grid                   |
| -------- | ---------------- | ---------------------- |
| 布局维度 | 一维（行或列）   | 二维（行 + 列）        |
| 场景     | 内容分布、对齐   | 整体结构、网格布局     |
| 优点     | 简单直观，兼容好 | 强大灵活，适合复杂页面 |
| 常用组合 | 组件内部         | 页面外层框架           |

## 🌟 资深前端可加分内容

### 不使用媒体查询即可实现响应式布局

让 Grid **自动根据容器宽度调整列数**,当容器变宽时，会自动多显示几列。当容器变窄时，列数会减少，直到卡片宽度不小于 250px。

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 16px;
}
```

### 嵌套布局自动对齐

问题：在嵌套 Grid 布局中，内层 Grid 默认无法继承外层的列宽或行高定义。这导致子元素很难对齐外层元素。

```css
.outer {
  display: grid;
  grid-template-columns: 150px 1fr 1fr;
  gap: 10px;
}

.inner {
  display: grid;
  grid-template-columns: subgrid;
  grid-column: 2 / 4; /* 占据父网格的第2到第3列 */
}
```
