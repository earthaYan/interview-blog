# margin 折叠

## 定义：

当两个垂直方向上的盒子相遇时，它们的 margin 可能会合并成一个外边距，这个现象称为 margin 重叠，他只会发生在普通文档流中，水平方向的 margin 不会产生折叠，脱离文档流的元素也不会发生折叠

## 取值：

- 如果都是正值 → 取较大的那一个
- 如果都是负值-> 两个 div 会重叠，重叠的距离就是较小的 margin 值
- 如果一正一负 → 取正负相加后的结果

## 发生范围

### 相邻兄弟元素之间

```html
<div class="a"></div>
<div class="b"></div>
<style>
  .a {
    margin-bottom: -30px;
    background: lightblue;
    height: 50px;
  }
  .b {
    margin-top: 50px;
    background: lightgreen;
    height: 50px;
  }
</style>
```
