# DTLE 项目(一)：项目简介

## 项目背景

DTLE 是一款数据传输工具，提供数据迁移、增量数据订阅和数据同步等能力。相比传统 ETL 工具，DTLE 支持多种异构数据源之间的数据传输，具备可视化管理界面和向导式链路创建流程，大幅降低了使用门槛。除此之外，平台还提供 nomad 节点管理、用户管理与权限管理等功能，帮助企业更高效、更安全地构建和运维数据集成链路。

## 技术选型

该项目的前端部分由我作为前端开发负责人的角色 从 0 到 1 独立完成，技术选型上：

- 项目框架：React
- 类型检查：TypeScript
- 路由系统：React-Router
- 全局状态管理：Redux-toolkit
- 代码规范:prettier,stylelint,eslint
- 脚手架：cra+craco
- http 请求库：axios
