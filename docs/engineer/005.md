# html 处理

## html 作用

1. 提供 html 模板，复用固定内容
2. 打包的时候生成一个 html
3. 打包出来的 html 自动引入打包后的 js/css

## webpack 处理

::: code-group

```js [单入口]
const htmlWebpackPlugin = require("html-webpack-plugin");
module.exports = {
  plugins: [
    new htmlWebpackPlugin({
      template: "./index.html",
      filename: "index.html",
    }),
  ],
};
```

```js [多入口]
const htmlWebpackPlugin = require("html-webpack-plugin");
module.exports = {
  plugins: [
    new htmlWebpackPlugin({
      template: "./index.html",
      filename: "index.html",
      chunks: ["app"],
      //html压缩配置
      minify: {
        collapseWhitespace: false,
        removeComments: false,
      },
      //script放置位置
      inject: "body", //body|true,head,false
    }),
    new htmlWebpackPlugin({
      template: "./index.html",
      filename: "index2.html",
      chunks: ["app2"],
      minify: {},
    }),
  ],
};
```

:::
