# 处理 js-根本

## 常用操作

1. es6 转换->babel-loader
2. 代码规范->eslint
3. 代码的分割和打包->webpack 自身核心功能

## babel-loader

1. 安装`npm i  --save-dev  babel-loader@8  @babel/core@7`
2. 安装`@babel/preset-env`
3. 将 babel 写入 webpack 配置，`loader` 值只能是一个字符串，而如果需要对 babel-loader 做一些配置，则必须使用 `use`

```js
module.exports = {
  module: {
    rules: [
      {
        test: /\.vue$/,
        loader: "vue-loader",
      },
      {
        test: /\.js$/,
        use: [
          {
            loader: "babel-loader",
            options: {
              presets: [
                [
                  "@babel/preset-env",
                  {
                    target: {
                      browsers: [">1%", "last 2 versions", "not ie<=8"],
                    },
                  },
                ],
              ],
            },
          },
          "xxx-loader",
        ],
      },
    ],
  },
};
```

4. 也可以将 babel-loader 配置写入单独的`.babelrc`文件

## eslint

1. 安装插件:`npm install --save-dev eslint eslint-webpack-plugin`
2. 注册插件:

```js
const eslintPlugin = require("eslint-webpack-plugin");

module.exports = {
  plugins: [new eslintPlugin()],
};
```

3. 定义配置:eslint8 通过`.eslintrc.js`配置，eslint9 不向下兼容 eslint8

```js
module.exports = {
  env: {
    browser: true,
    node: true,
    es2021: true,
  },
  //现成的规范-通过npm包安装 eslint-config-standard eslint-config-airbnb
  extends: ["standard", "airbnb"],
  // 特殊语法的支持，比如vue
  plugins: [],
  parserOptions: {},
  rules: {
    // 0-off 1-warn 2-error
    "no-console": 0,
  },
};
```
